**1.CRITICAL RISK (Score: 1.0)**

**Title: Apache Tomcat Request Smuggling Vulnerability**

**CVE: CVE-2022-42252**

**Description: A request smuggling vulnerability was discovered in Apache Tomcat. If Tomcat is configured to ignore invalid HTTP headers, a malicious proxy or intermediary can bypass security controls. This actively exploited vulnerability specifically affects Tomcat version 9.0.65. Administrators must patch immediately, especially for internet-facing application servers.**



**2.HIGH RISK (Score: 0.50)**

**Title: VMware ESXi Authentication Bypass**

**CVE: CVE-2024-37085**

**Description: A critical authentication bypass vulnerability exists in the VMware ESXi hypervisor. A malicious actor with sufficient network access to the management interface can bypass Active Directory authentication and gain full administrative privileges. This vulnerability affects VMware ESXi version 8.0 and requires immediate remediation.**



**3.LOW RISK (Score: 0.0)**

**Title: Palo Alto Networks PAN-OS GlobalProtect Command Injection**

**CVE: CVE-2024-3400**

**Description: A critical command injection vulnerability in the GlobalProtect feature of Palo Alto Networks PAN-OS software for specific PAN-OS versions enables an unauthenticated attacker to execute arbitrary code with root privileges on the firewall. This issue affects PAN-OS 10.2, PAN-OS 11.0, and PAN-OS 11.1 firewalls.**



**4.LOW RISK (Score: 0.0)**

**Title: Citrix NetScaler ADC and Gateway "Citrix Bleed" Vulnerability**

**CVE: CVE-2023-4966**

**Description: A critical vulnerability in Citrix NetScaler ADC and NetScaler Gateway allows for unauthenticated sensitive information disclosure. Attackers can hijack existing authenticated sessions, bypassing multifactor authentication. Organizations running vulnerable Citrix appliances must upgrade immediately to prevent unauthorized access to internal networks.**
